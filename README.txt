🎌 AnimeFinder - Descubre tu Próximo Anime Favorito
=====================================================

Una aplicación web moderna y responsiva construida con JavaScript vanilla que ayuda a los entusiastas del anime a descubrir nuevas series y obtener recomendaciones personalizadas.

✨ CARACTERÍSTICAS PRINCIPALES
==============================

• Búsqueda Inteligente: Búsqueda de anime en tiempo real con sugerencias de autocompletado
• Recomendaciones Inteligentes: Motor de recomendaciones basado en géneros y similitud
• Soporte Bilingüe: Soporte completo en inglés y español con detección automática
• Diseño Responsivo: Hermoso tema oscuro optimizado para todos los dispositivos
• Información Detallada: Detalles completos del anime incluyendo sinopsis, calificaciones, géneros y estudios
• UI/UX Moderna: Animaciones suaves, efectos hover y navegación intuitiva

🛠️ STACK TECNOLÓGICO
=====================

• Frontend: HTML5, CSS3, JavaScript (ES6+)
• API: Jikan API (MyAnimeList)
• Estilos: CSS personalizado con CSS Grid y Flexbox
• Iconos: Font Awesome
• Fuentes: Inter (Google Fonts)

🚀 FUNCIONALIDADES
==================

1. BÚSQUEDA DE ANIME
   - Búsqueda en tiempo real con debounce
   - Sugerencias de autocompletado
   - Filtrado por contenido seguro (SFW)

2. SISTEMA DE RECOMENDACIONES
   - Recomendaciones basadas en géneros
   - Recomendaciones por similitud
   - Algoritmo inteligente de matching

3. INTERNACIONALIZACIÓN
   - Detección automática del idioma del navegador
   - Cambio manual de idioma (ES/EN)
   - Persistencia de preferencias en localStorage

4. INTERFAZ DE USUARIO
   - Tema oscuro moderno
   - Diseño responsivo para móviles y desktop
   - Animaciones suaves y transiciones
   - Modales informativos

🔧 INSTALACIÓN Y USO
====================

1. Clona o descarga el repositorio
2. Abre index.html en tu navegador web
3. ¡Disfruta descubriendo nuevos animes!

No se requiere instalación de dependencias adicionales.

🌟 CARACTERÍSTICAS TÉCNICAS
============================

• Arquitectura modular con separación de responsabilidades
• Manejo de errores robusto con notificaciones al usuario
• Optimización de rendimiento con lazy loading de imágenes
• Sistema de caché para mejorar la experiencia del usuario
• API rate limiting y manejo de límites de la API externa

📊 API UTILIZADA
================

Jikan API (https://jikan.moe/)
- API no oficial de MyAnimeList
- Gratuita y sin límites estrictos
- Proporciona datos completos de anime y manga
